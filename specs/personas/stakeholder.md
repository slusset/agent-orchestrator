# Persona: Stakeholder

## Role
Product owner or technical lead who requests features and tracks progress. Communicates with the Primary Agent as their main interface to the development pipeline.

## Goals
- Request features in natural language and have them implemented
- Track progress without micromanaging agents
- Get notified when work is blocked or needs decisions
- Approve deployments to production

## Frustrations
- Having to context-switch between JIRA, GitHub, Slack to track progress
- Not knowing if work is stuck until it's too late
- Having to manually coordinate handoffs between dev, QA, and ops

## Context
- Tech comfort: high
- Usage frequency: daily
- Key devices: desktop (primary), mobile (notifications)

## Quotes
> "I want to describe what I need and have it show up as a PR I can review."
> "Don't surprise me — tell me when something's blocked before it becomes a crisis."
